# tmux-mgr

A TUI for managing, navigating, and launching `tmux` projects and sessions.

`tmux-mgr` replaces the manual overhead of creating and switching between tmux sessions with an intuitive, fuzzy-searchable interface. It tracks your favorite projects, supports dynamic folder scanning, and integrates seamlessly with [`tmux-resurrect`](https://github.com/tmux-plugins/tmux-resurrect).

## Installation

Ensure you have Go installed, GOPATH is set and $GOPATH/bin is in your $PATH, then run:

```bash
go install github.com/justanoobcoder/tmux-mgr@latest
```
 
Or build from source:

```bash
git clone https://github.com/justanoobcoder/tmux-mgr.git
cd tmux-mgr
go build .
```
Then copy the binary to your `$PATH`.
Or download the binary file 

## Dependencies
- `tmux` must be installed and available in your `$PATH`.
- (Optional but recommended) `tmux-resurrect` for session restoration.

## Usage

You can launch the main interactive project picker simply by running:

```bash
tmux-mgr
```

### Commands

* **`tmux-mgr projects`** (or just `tmux-mgr`)
  Opens the TUI project picker.

* **`tmux-mgr sessions`**
  Opens the TUI session picker.

* **`tmux-mgr add [path]`**
  Adds a specific project directory to your tracker. If no path is provided, it adds your current working directory.

* **`tmux-mgr add --folder [path]`**
  Adds a parent folder. All subdirectories inside this folder will automatically show up as selectable projects. 
  *You can use `--exclude [name]` to ignore specific subdirectories (e.g., `node_modules`).*

* **`tmux-mgr remove [path]`**
  Removes a configured project or folder.

* **`tmux-mgr config`**
  Opens your configuration file in your default `$EDITOR`.
  
* **`tmux-mgr config --editor <editor>`**
  Opens your configuration file in `<editor>`.
  
* **`tmux-mgr config init`**
  Create an init configuration file in `~/.config/tmux-mgr/config.json`.

* **`tmux-mgr config show`**
  Show configuration content.

### TUI Keybindings

While in the `projects` or `sessions` interface:
- **`j` / `k`** or **`Up` / `Down`**: Navigate the list.
- **`/`**: Start fuzzy searching.
- **`Esc`**: Toggle between Fuzzy search and Exact match search.
- **`Enter`**: Select and attach to the session.
- **`d`**: Prompt to delete the highlighted project or kill the highlighted session.
- **`q` / `Ctrl+C`**: Quit.

## Configuration

The configuration file is automatically created at `~/.config/tmux-mgr/config.json` on the first run.
Example:
```json
{
  "projects": [
    "/home/user/my_app"
  ],
  "folders": [
    {
      "path": "/home/user/workspaces",
      "excludes": ["node_modules", ".git"]
    }
  ],
  "tmux": {
    "session_prefix": "",
    "attach_on_create": true
  },
  "resurrect": {
    "enabled": true,
    "save_dir": "/home/user/.tmux/resurrect"
  },
  "scores": {
    "/home/user/my_app": 5
  }
}
```

* **`tmux.session_prefix`**: A string prefixed to all tmux session names generated by `tmux-mgr`.
* **`tmux.attach_on_create`**: Set to `false` if you want `tmux-mgr` to launch sessions in the background without automatically attaching your terminal to them.
* **`resurrect.enabled`**: Set to `false` to disable integration with `tmux-resurrect`.
* **`scores`**: Automatically managed tracking metrics that sort your most frequently opened projects to the top.
